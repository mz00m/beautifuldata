# Competitor Analysis: FigureLabs.ai

_Date: March 2026_

## What FigureLabs Does

FigureLabs (figurelabs.ai) is an **AI-powered scientific illustration tool** founded by Oxford University alumni (launched August 2025). It automates the creation of publication-quality scientific figures for researchers and academics.

### Core Features

| Feature | Description |
|---|---|
| **Sketch-to-Figure** | Transforms hand-drawn sketches into precise, professional scientific images using AI |
| **Text-to-Figure** | Extracts data from research papers/PDFs to generate figures; also generates images from text descriptions |
| **Figure Refiner** | Enhances existing figures for publication (resolution, color correction, noise reduction) |
| **Image Vectorization** | Converts raster images to scalable vector graphics |

### Target Market

- Academic researchers (biology, engineering, etc.)
- Students and early-career researchers
- Scientists preparing journal submissions
- Anyone needing publication-quality scientific figures

### Pricing

FigureLabs' specific pricing tiers are **not publicly indexed** as of March 2026. Their pricing page exists but is not crawlable, suggesting either a gated/waitlist model or very recent launch pricing. This is worth monitoring.

### Technology

Leverages machine learning, NLP, and computer vision. Emphasizes accessibility — no design expertise required.

## How Beautiful Data Differs

### Fundamental Positioning Difference

| Dimension | FigureLabs | Beautiful Data |
|---|---|---|
| **Core product** | AI tool for static scientific figures | Studio for interactive data experiences |
| **Output format** | Static images (PNG, SVG) for journals | Live, interactive web applications |
| **Delivery model** | Self-service SaaS tool | Bespoke studio service (fixed-price projects) |
| **User interaction** | User operates the tool | Studio builds for the client |
| **End audience** | Journal reviewers / academic peers | Public audiences, stakeholders, decision-makers |
| **Data complexity** | Single-figure, single-dataset | Multi-source, narrative-driven, cross-dataset |

### Where Beautiful Data Wins

1. **Interactivity is the product, not an afterthought**
   - FigureLabs outputs static PNGs/SVGs. Beautiful Data builds scrollytelling, explorable dashboards, and interactive charts that audiences *engage* with rather than glance at.

2. **Narrative-driven data storytelling**
   - FigureLabs helps make a figure. Beautiful Data tells the *story* — guiding users through complex data with scrollytelling, progressive disclosure, and context.

3. **Verified, multi-source evidence systems**
   - The 4-tier evidence system (demonstrated in jobsdata.ai with 300+ sources) goes far beyond rendering a chart. This is a credibility layer that FigureLabs doesn't attempt.

4. **Production-grade web experiences**
   - Not embedded Tableau, not a static image — real Next.js web applications deployed to client domains with no licensing fees.

5. **Higher-value engagements**
   - $5k–$20k+ project pricing vs. likely $10–50/mo SaaS. Different markets, but Beautiful Data captures far more value per client.

### Where FigureLabs Wins (Be Honest About It)

1. **Speed & self-service** — Researchers get figures in seconds, no project scoping needed.
2. **Volume** — Can serve thousands of researchers simultaneously; Beautiful Data is capacity-constrained.
3. **Lower barrier to entry** — Likely a free tier or low monthly cost vs. $5k minimum engagement.
4. **Academic-specific features** — Journal formatting, resolution standards, and publication conventions built in.

## Strategic Implications

### They Are NOT a Direct Competitor

FigureLabs and Beautiful Data serve fundamentally different needs:

- **FigureLabs** = "Help me make a figure for my paper" (tool)
- **Beautiful Data** = "Help me turn my research into something the world can explore" (service)

The overlap is thin: both deal with data visualization. But the similarity ends there.

### Potential Threat Vectors

1. **Feature creep upstream**: If FigureLabs adds interactive/web export capabilities, they could eat into the Starter tier ($5k single-visualization projects).
2. **Market education**: FigureLabs may train researchers to expect AI-generated visuals at near-zero cost, making the $5k entry point feel steep.
3. **Partnership opportunity**: Researchers who use FigureLabs for papers may need Beautiful Data when they want to *publicize* their findings interactively.

### Recommended Differentiation Messaging

> "FigureLabs makes figures. We make experiences."

Key angles to emphasize in positioning:

- **"Your research deserves more than a figure"** — extend the existing "more than a PDF" tagline
- **Interactive, not static** — always lead with the fact that outputs are live web experiences
- **Guided narrative** — scrollytelling is something AI figure tools fundamentally cannot do
- **Source verification** — the evidence tier system is a unique trust layer
- **No ongoing licensing** — ship to your domain, you own it forever
